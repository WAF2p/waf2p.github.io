Text
Freigegebener Boilerplate-Text

Folgender Text kann bei der Beschreibung von WAF++ in Artikeln, Podcasts oder Präsentationen verwendet werden.

Kurz (1 Satz)

WAF++ ist ein Open-Source-, anbieterneutrales Framework zur Bewertung und Verbesserung der Qualität von Cloud-Workloads über 7 Säulen: Sicherheit, Zuverlässigkeit, Performance-Effizienz, Kostenoptimierung, operationelle Exzellenz, Nachhaltigkeit und Developer Experience.

Standard (2–3 Sätze)

WAF++ (Well-Architected Framework Plus Plus) ist ein Open-Source-, community-gesteuertes Framework zur Bewertung und Verbesserung der Qualität von Cloud-Workloads. Es umfasst 7 Säulen — Sicherheit, Zuverlässigkeit, Performance-Effizienz, Kostenoptimierung, operationelle Exzellenz, Nachhaltigkeit und Developer Experience — mit einem transparenten Scoring-Modell namens PASS.

Im Gegensatz zu proprietären Anbieter-Frameworks wird WAF++ offen über RFCs in GitHub Discussions gesteuert, sodass jede Entscheidung nachvollziehbar und jeder Control zitierbar ist. Das Framework ist kostenlos nutzbar, kann frei beigetragen und weiterentwickelt werden.
