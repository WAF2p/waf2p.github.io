About
WAF++ in brief

WAF++ (Well-Architected Framework Plus Plus) is an open-source, vendor-neutral framework for assessing and improving the quality of cloud workloads. Built by a community of cloud engineers and architects, WAF++ extends cloud provider well-architected reviews with a transparent, community-driven scoring model — the PASS model — across 7 pillars: Security, Reliability, Performance Efficiency, Cost Optimisation, Operational Excellence, Sustainability, and Developer Experience.

Unlike proprietary frameworks, WAF++ is governed openly via GitHub Discussions and RFCs. Every decision is traceable, every control is documented, and every contribution is welcome — from individual engineers to platform teams at scale.

The project is hosted at github.com/waf2p and the documentation is published at waf2p.github.io.
